Source Radiation Measurements
26 January 2026
8:56

Aki (1967) and Brune (1970)
Source radiation --\> frequency-dependent energy emitted by the earthquake source
Low frequency = seismic moment
High frequency = corner frequency, stress drop --\> physical dimension of fault

Tucker & Brune 1977
To determine relative or absolute size of earthquakes, spectral measurements on direct P or S need to be corrected for:
1.  Source radiation pattern (nonisotropic)
2.  Propagation radiation effects
--\> difficult to make

S coda is an incoherent superposition of scattered S wave from all directions
--\> no need to know source radiation pattern or propagation effects

![image1](1ff8abf677864ccc8f2f7aa339068bab.png)
Dividing coda amplitude of seismogram at site j for earthquake i and reference earthquake j
The ratio of late S-coda energies equals the ratio of source radiated energies.
Same lapse time --\> same transport history
Same station --\> same site response
Same frequency --\> same attenuation
Thus these parameters cancel

Single station is sufficient because coda is spatially uniform at late times

Practical Steps
1.  Event selection
2.  Choose reference event
    1.  Well recorded
    2.  Moderate size
    3.  Representative of the dataset
3.  Preprocess waveform (IRC)
4.  Bandpass filter
5.  Determine Ts and lapse time (same for all events)
6.  Define coda window
7.  Compute coda energy (MS or RMS)
8.  Source radiation ratios
9.  Stack over multiple coda windows
